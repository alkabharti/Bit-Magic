Time Complexity :


```java
public static int getFirstSetBit(int n)
{
    int c=0;
    loop:
    while(n>0)
    {
        c++;
        if(n%2==1)
        {
            break loop;
        }
        n=n/2;
    }
    return c;    
}
```
