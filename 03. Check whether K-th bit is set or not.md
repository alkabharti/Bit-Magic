Solution 1 :

Time Complexity :


```java
static boolean checkKthBit(int n, int k)
{
    int c=0;
    loop:
    while(n>0)
    {
        if((n & 1)==1 && c==k)
        {
            return true;
        }
        c++;
        n=n>>1;
    }
    return false;
}
```

Solution 2 :

Time Complexity :


```java
static boolean checkKthBit(int n, int k)
{
    String str=Integer.toBinaryString(n);
    if(str.charAt(str.length()-1-k)=='1')
        return true;
    return false;
}
```
