Solution :

Time Complexity :


```java
public static int posOfRightMostDiffBit(int m, int n)
{
    int xor=m^n;
    int i,res=-1,c=0;
    while(xor>0)
    {
        c++;
        if(xor%2==1)
        {
            res=c;
            break;
        }
        xor=xor/2;
    }
    return res;
}
```
