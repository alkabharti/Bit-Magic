Time Complexity :


```java
public static int countSetBits(int n)
{
    if(n<=1) 
        return n;

    int pow = 0;
    while((1 << pow)<=n)
    {
        pow+=1;
    }
    pow-=1;

    return (1 << pow-1)*pow + (n-(1 << pow)+1) + countSetBits(n-(1<<pow));

}
```
