### Solution 1 :

- Time Complexity : 


```java
static Long numberOfSubsequences(int N, ArrayList<Long> A)
{
    int count=0;
    long M=1000000000+7;
    for(int i=0;i<A.size();i++)
    {
        if((A.get(i) & (A.get(i)-1))==0)
        {
            count++;
        }
    }
    return (long) (Math.pow(2,count)%M-1)%M;
}

```
